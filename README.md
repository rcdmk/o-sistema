# O Sistema

This project was one of the final projects in college, back in 2009. It is a platformer game written in C# using the extinct XNA framework 3.1 for .Net Framework 3.5. The original solution was built in Visual Studio 2008, with C# 3.0 and ported to dotnet 8 on MonoGame 3.8 in 2025.
